# template-project
The standard boilerplate for METU IEEE CS software projects. Includes standard directory structure, Docker setup, and CI/CD workflows.
*important note: there are three README badges on top of this readme, the middle one is (which is about the **Build Status**) dynamic therefore you dont have to change it when you set up once. However, it has to be configured for every repository. Therefore, you need to open the readme and change the link (**DON'T CHANGE IT ON THIS TEMPLATE**) whenever you create a new repository in order to configure properly.*
